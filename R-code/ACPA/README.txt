ACPA.R uses the AMPA groups for analysis and also contains code to identify new groups.

ACPA2.R only contains the code to identify new groups since AMPA groups were missing for a lot of patients.

All the files related to MPlus can be found in MPlus_files.

Number_of_positives-1.csv contains ACPA fine specificity data with number of positives (from cutoff calculation) for each patient and their assigned AMPA groups.
