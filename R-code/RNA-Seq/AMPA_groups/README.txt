AMPA.R contains the code for DESeq2 analysis. The files  after running AMPA.R can be found in this folder

AMPA_results.R contains the code for all the plots of significant genes including heatmap. This code uses the files generated by AMPA.R

Deseq.R and Results.R is for comparison amongst the AMPA group members.DESeq-2.R is for negative vs triple comparison.
